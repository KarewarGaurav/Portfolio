# Directives (Layer 1)

This directory contains Standard Operating Procedures (SOPs) for the Antigravity agent.

## Core Purpose
Directives define **what** needs to happen in a deterministic way. They act as the "instruction manual" for the agent to follow when performing complex or repetitive tasks.

## How to use
1. **Define Intent**: Create a new `.md` file here for any complex automation.
2. **Deterministic Steps**: List the logic, inputs, and expected outputs.
3. **Link to Execution**: Specify which scripts in the `execution/` folder should be called to perform the work.

Refer to [GEMINI.md](../../GEMINI.md) for the full architectural rules.
