# 🗺️ Portfolio Content Map

This document serves as a guide to help you quickly find and update the content of your portfolio. Use this map to edit text, images, or links without having to search through the entire codebase.

---

## 🚀 Main Sections

| Section | File Path | What's inside? |
| :--- | :--- | :--- |
| **Hero** | `src/components/sections/Hero.tsx` | Your name, headline, social links, and Resume button. |
| **About** | `src/components/sections/About.tsx` | Your biography and "About Me" description. |
| **Experience** | `src/components/sections/Experience.tsx` | Job history, company names, and project roles. |
| **Projects** | `src/components/sections/Projects.tsx` | List of your work, descriptions, tech tags, and project images. |
| **Skills** | `src/components/sections/Skills.tsx` | Technical skills, tools, and category badges. |
| **Certifications** | `src/components/sections/Certifications.tsx` | Your earned certificates and achievements. |
| **Education** | `src/components/sections/Education.tsx` | Degrees, college info, and academic history. |
| **Contact** | `src/components/sections/Contact.tsx` | Email form and contact information. |

---

## 🖼️ Images & Assets

All static files are located in the `public/` directory:
- **Resume**: `public/Resume AI.pdf`
- **Avatar/Profile Picture**: `public/avatar.jpg`
- **Project Previews**: `public/` (e.g., `astraclean-preview.png`, `vitrag-preview.png`)

## ⚙️ Configuration & Data

| Purpose | File Path |
| :--- | :--- |
